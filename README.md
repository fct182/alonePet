# alonepet

## Project detail

**项目介绍：**

独行Pet之家，移动端Web应用。展示周围流浪狗、可对流浪狗进行投喂、领养。也可新发布你发现的流浪狗，通过上传宠物狗犬鼻纹，犬鼻纹可成为狗狗的身份证。通过犬鼻纹判断两只狗是否为同一只。

**实现技术：**

Vue全家桶+Vant组件库



**当前为应用的前端应用**
